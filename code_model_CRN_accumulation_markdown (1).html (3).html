Reduced chi-squared function ¶

def redchisqg(ydata,ymod,deg=10,sd=None):

  if sd[0]==None:
       chisq=np.sum((ydata-ymod)**2)
  else:
       chisq=np.sum( ((ydata-ymod)/sd)**2 )

  nu=ydata.size-1-deg
  return chisq/nu

CRN accumulation over depositional and post depositional history ¶

“”“t_fin, Erosion_Burial and N_inh are independant lists (type values between brackets, as for classical lists). t_fin
is a list of time periods in years. Exposure_age is the sum of all the time periods specified in the list t_Fin, Erosion_burial is a list of denudation or aggradation values in cm. By convention, erosion is positive and aggradation negative. A value of 0 means landscape stability. In case of aggradation period, the amount of centimeters the user wishes to aggrade must be preceded by a minus sign. t_fin and Erosion_Burial lists MUST have the same length. N_inh is a list of inherited 10Be concentrations. The first term of the list is the inherited concentration of the base column. the other terms are the inherited concentrations of the material accumulated during each aggradation step. N_inh length is thus one plus the amount of aggradation phases.Exposure age remaining equals exposure age at t0 and will decrease as we move from one period to another. base_thickness is the thickness of the sediment column at tO (cm). The denudation value at any time, and the difference between total denudation and total aggradation, must be lower than the base column thickness. Adapt the base_thickness default value (1500 cm) if needed, otherwise the base column will become negative and make the model crash”""
